# Security Policy

## Supported versions

USB Guardian is pre-release software. Security fixes are applied to the latest published beta only.

| Version | Supported |
| --- | --- |
| 0.1.0-beta2 | Yes |
| 0.1.0-beta1 | No |

## Reporting a vulnerability

Use GitHub's **Security > Report a vulnerability** form to send a private report. Include the affected version, relevant Unraid and Unassigned Devices versions, the security boundary that was crossed, reproduction conditions, and a minimally redacted diagnostic bundle when available.

Do not open a public issue containing exploit instructions, credentials, private keys, access tokens, or sensitive diagnostic data. A public issue may be used for non-sensitive safety concerns that cannot be exploited.
